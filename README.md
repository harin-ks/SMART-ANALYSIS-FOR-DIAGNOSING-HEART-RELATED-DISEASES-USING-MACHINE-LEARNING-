# SMART-ANALYSIS-FOR-DIAGNOSING-HEART-RELATED-DISEASES-USING-MACHINE-LEARNING-
Smart Analysis for Diagnosing Heart Related Diseases Using Machine Learning is a healthcare ML project that predicts heart disease risk using patient data. It compares algorithms like Random Forest, SVM, XGBoost, Gradient Boosting, and ANN with preprocessing, PCA, and evaluation techniques for accurate early diagnosis.
# ❤️ Smart Analysis for Diagnosing Heart Related Diseases Using Machine Learning

## 📌 Overview

This project focuses on predicting heart-related diseases using Machine Learning techniques. The system analyzes patient health data and classifies whether a person is at risk of heart disease. Multiple machine learning algorithms were implemented and compared to identify the most accurate prediction model.

The project aims to support early diagnosis and improve healthcare decision-making through intelligent data analysis.

---

## 🎯 Objectives

* Predict heart disease using patient medical data
* Compare the performance of different ML algorithms
* Improve prediction accuracy using preprocessing and PCA
* Analyze model performance using evaluation metrics

## 📂 Dataset Information

The dataset contains patient health records with features such as:

* Age
* Gender
* Blood Pressure
* Cholesterol Level
* Heart Rate
* Diabetes
* Smoking
* BMI
* Physical Activity
* Family History

### Target Variable

* `1` → Heart Disease Present
* `0` → No Heart Disease

## 📊 Evaluation Metrics

The models were evaluated using:

* Accuracy

* Recall
* F1-Score
* Confusion Matrix

---

## 📈 Results

* Random Forest, SVM, XGBoost, and Gradient Boosting achieved high prediction accuracy.
* PCA improved model efficiency and reduced dimensionality.
* ANN required more tuning for better performance.
