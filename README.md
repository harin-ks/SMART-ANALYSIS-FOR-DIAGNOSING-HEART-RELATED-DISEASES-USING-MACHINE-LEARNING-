# SMART-ANALYSIS-FOR-DIAGNOSING-HEART-RELATED-DISEASES-USING-MACHINE-LEARNING-
Smart Analysis for Diagnosing Heart Related Diseases Using Machine Learning is a healthcare ML project that predicts heart disease risk using patient data. It compares algorithms like Random Forest, SVM, XGBoost, Gradient Boosting, and ANN with preprocessing, PCA, and evaluation techniques for accurate early diagnosis.
